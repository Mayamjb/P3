Maze game performed as part of the third project of the application developer course - Python.
Aim of the game: Collect three objects, the needle, the pastic tube and the ether, 
to unlock the syringe, put the guard to sleep and be able to get out of the labyrinth.


This game uses python 3.7.3 and pygame 1.9.6
